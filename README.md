# wine_calculator_app
# 🍷 와인 양조 계산기 웹앱

Streamlit 기반의 **와인 양조 보조 웹앱**으로, 누구나 쉽게 당도 계산 및 첨가제 투입량을 계산할 수 있습니다.  
초보 홈브루어부터 소규모 양조장까지 폭넓게 활용 가능한 도구입니다.

---

## 🧩 주요 기능

### 1. 📈 브릭스(Brix) 당도 계산기

- 기준 용액의 양(L), 현재 브릭스, 목표 브릭스를 입력하면  
  **필요한 당류의 투입량(g)**을 자동 계산합니다.
- 선택 가능한 당류 종류:
  - 백설탕 (100%)
  - 꿀 (79%)
  - 포도당 시럽 (70%)

### 2. ⚗️ 첨가제 투입량 계산기

- 전체 용액 부피(L)를 기준으로 다음 4가지 양조 첨가제의 투입량을 계산합니다:
  - 아황산염 (KMS) → 0.1g/L
  - 효모 → 0.2g/L
  - 제2인산암모늄 (DAP) → 0.3g/L
  - 소르빈산염 → 0.2g/L
